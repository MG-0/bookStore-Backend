# Bookstore API

A backend RESTful API built with Node.js and Express.js for managing a bookstore system. The application provides endpoints to create, read, update, and delete books while following REST API principles and clean project structure.

## Features

* 📚 Manage bookstore inventory
* ➕ Create new books
* 🔍 Retrieve book details
* ✏️ Update existing books
* ❌ Delete books
* 🌐 RESTful API architecture
* ⚡ Express.js middleware support

## Tech Stack

* Node.js
* Express.js
* JavaScript
* REST API
* MongoDB

## Purpose

This project was built to practice backend development fundamentals, API design, request handling, routing, middleware usage, and CRUD operations using Node.js and Express.js.
